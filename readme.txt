== Description ==

The Elegant Zazzle Plugin (E.Z.P.) allows you to easily display Zazzle products on your WordPress site one of two ways:
* via Wordpress shortcodes in a blog post or page.
* via widget in a widget area.

With this plugin, you can easily advertise your Zazzle products on your Wordpress site. Zazzle is free to join and easy to use.

= To use via shortcode =
Enter some text like this into any page or a post:
<code>[ezp grid_width="600" grid_cell_size="medium" grid_cell_spacing="40" num_items="3" store_name="lupitafdezsoberon"]</code>
This will show 3 products, each of medium size and 40 pixels apart, from the Zazzle store <a href="http://www.zazzle.com/lupitafdezsoberon?rf=">lupitafdezsoberon</a> in a grid that is 600 pixels wide. All the configuration options are listed <a href="http://www.zazzletools.com/elegant-zazzle-plugin/">here</a>.

= To use via widget =
Goto to Appearance -> Wigets and drag and drop the Zazzle Widget to any widget area supported by your theme. All the configuration options are listed <a href="http://www.zazzletools.com/zazzle-widget/">here</a>.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload files to wp-content/plugins/zazzle-plugin
2. Activate it through the plugin management screen.
